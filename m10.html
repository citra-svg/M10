Blog Praktikum Stored Procedure Lanjutan- SQL

Pada praktikum ini, dipelajari penggunaan Stored Procedure mulai dari parameter IN, OUT, IF/ELSE, JOIN, GROUP BY, hingga operasi DML seperti INSERT, UPDATE, dan DELETE.

Contoh Query 1 - IF ELSE Status Mahasiswa

CREATE PROCEDURE showByKategori(IN p_kategori VARCHAR(20))
BEGIN
    IF p_kategori = 'aktif' THEN
        SELECT * FROM mahasiswa WHERE idstatusaka = 1;
    ELSE
        SELECT * FROM mahasiswa WHERE idstatusaka = 4;
    END IF;
END;

CALL showByKategori('aktif');

Query ini digunakan untuk menampilkan data mahasiswa berdasarkan kategori status menggunakan IF/ELSE.

Contoh Query 2 - OUT Parameter AVG IPK

CREATE PROCEDURE avgIpkStatus(IN p_status INT, OUT p_avg REAL)
BEGIN
    SELECT ROUND(AVG(ipk),2)
    INTO p_avg
    FROM mahasiswa
    WHERE idstatusaka = p_status;
END;

CALL avgIpkStatus(1,@hasil);
SELECT @hasil;

Query ini digunakan untuk menghitung rata-rata IPK berdasarkan status mahasiswa menggunakan parameter OUT.

Kesimpulan

Stored Procedure membantu menyederhanakan query, meningkatkan efisiensi, dan membuat logika database lebih terstruktur.
